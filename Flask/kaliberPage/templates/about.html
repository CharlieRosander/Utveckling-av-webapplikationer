About KaliberTestPage


This is a website made for a project, and it's purpose is educational and testing.

The website is written in HTML, using Flask and CSS/Bootstrap for styling/function.

The app creates a database, which stores the user information provided in the sign up/log in page. The password is also hashed.


The 2 APIs integrated with the webiste are OpenWeatherMap and OpenAI's ChatGPT.

The API-keys have to be provided by the user, and they are private and stored in a secure place.


The logo used on this website is AI-generated using Midjourney
